Punto 1
  1. ¿Cuando es conveniente utilizar cada medida?

En el caso de la media, que está siendo influenciada por todo el conjunto de datos incluidos los extremos, es necesario y conveniente sacar el resultado, recortando un porcentaje utilizando trim, para evitar incluir los datos extremos y que no afecten en gran medida el resultado general. La media recortada se calcula al arreglar los valores de la muestra en orden, “recortar” un número igual a partir de cada extremo y calcular la media de los restantes.

En el caso de la mediana, para recoger sin excepcion todos los elementos del conjunto, se deberia separar cada conjunto de clase para sacar su mediana respectiva, para que se pueda analizar cada imformacion del conjunto de elementos

En el caso de la moda, es conveniente cuando el conjunto de elementos ya sea variable o categoria, tiene más frecuencia en una muestra

Cuando se analizan datos categóricos, la moda es el único dato de tendencia central que puede utilizarse. Finalmente, la moda puede usarse como una medida de tendencia central para datos numéricos empleados en sentido categórico.

Punto 2
  1. ¿Qué tipo de variables son más adecuadas para ser representadas en una tabla de frecuencia para datos no agrupados?

Las variables cualitativas y cuantitativas se pueden presentar en una tabla de frecuencias para datos no agrupados, pero dependiendo del conjunto de elementos, esto puede ser mas conveniente cambiar el tipo de tabla:

  1. Si el conjunto de datos cuantitativos es bastante extensa, es mas conveniente utilizar una tabla de frecuencia para datos agrupados

  2. Para un conjunto de elementos cualitativos seria conveniente utilizar una tabla de frecuencia para datos no agrupados

  1. ¿Cuál es la diferencia principal entre una tabla de frecuencia para datos no agrupados y una para datos agrupados?

La tabla de frecuencia para datos agrupados presenta una agrupacion del conjunto de datos en la tabla, siguiendo la formula de la division para hallar la amplitud (el rango entre el total de clases). Para datos agrupados la frecuencia depende de la amplitud

  1. Explica la importancia de mantener la misma amplitud de clase en una tabla de frecuencia para datos agrupados.

Su importancia radica en que si no se utiliza, principalmente va a tener una distribucion no igualitaria entre el conjunto de datos, dando resultados que van a variar y no representar correctamente el conjunto de datos
